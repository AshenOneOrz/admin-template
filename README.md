## 使用

### clone

```bash
git clone https://github.com/AshenOneOrz/admin-template.git
```

### 安装依赖

```bash
yarn install
```

### 运行

```bash
yarn serve
```

### 打包 && lint

```bash
yarn build
yarn lint
```
